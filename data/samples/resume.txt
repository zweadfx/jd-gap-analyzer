백엔드 API 개발과 LLM 애플리케이션 프로토타이핑 경험이 있는 개발자입니다.
Python 기반 서비스 개발을 주로 해왔고, 최근 1년은 LLM을 붙인 내부 도구를 만들었습니다.

■ 경력

[가상컴퍼니] AI 프로덕트 팀 / 백엔드 엔지니어 (2024.03 ~ 현재)
- FastAPI로 4개 에이전트 API를 구현하고 사내 도구에 연동했습니다.
- OpenAI API를 사용해 문서 요약 기능을 만들었고, 프롬프트를 반복 튜닝해 사용자 만족도를 개선했습니다.
- Pydantic으로 LLM 구조화 출력을 정의하고, 파싱 실패 시 재시도하는 로직을 작성했습니다.
- PostgreSQL 스키마를 설계하고 SQLAlchemy로 ORM 레이어를 작성했습니다.
- Ruff와 pre-commit을 도입해 팀의 코드 스타일 논쟁을 없앴습니다.
- 주니어 개발자 1명의 온보딩을 맡아 3개월간 코드 리뷰를 진행했습니다.

[개인 프로젝트] 채용 공고 갭 분석기 (2026.07 ~)
- 채용 공고와 이력서를 비교해 근거 없는 요구사항을 찾아주는 CLI를 만들고 있습니다.
- LLM이 인용문을 지어내는 문제를 원문 대조 검증으로 잡았습니다.
