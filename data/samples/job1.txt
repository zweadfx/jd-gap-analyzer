[샘플 — 가상 공고] 핀테크A / LLM 플랫폼 백엔드 엔지니어

■ 팀 소개
사내 모든 프로덕트가 사용하는 LLM 게이트웨이와 에이전트 런타임을 만듭니다.
하루 수천만 건의 LLM 호출이 이 팀의 인프라를 지나갑니다.

■ 이런 일을 합니다
- LLM 게이트웨이 API 설계 및 운영
- 에이전트 실행 런타임 개발
- 모델 응답 품질 모니터링 파이프라인 구축

■ 자격요건
- Python 백엔드 개발 경험 3년 이상
- FastAPI 또는 그에 준하는 비동기 웹 프레임워크 사용 경험
- LLM 기반 서비스를 실제 프로덕션에 배포하고 운영한 경험
- Kubernetes 환경에서 서비스를 배포하고 운영한 경험
- 대용량 트래픽(일 100만 요청 이상)을 처리하는 시스템을 다뤄본 경험
- 관측 가능성(로깅, 메트릭, 트레이싱) 도구를 직접 구축하거나 운영한 경험

■ 우대사항
- vLLM, TGI 등 추론 서버를 직접 운영해본 경험
- RAG 파이프라인을 설계하고 성능을 개선해본 경험
- 금융 도메인 서비스 개발 경험
- 오픈소스 기여 경험
- 주니어 엔지니어를 멘토링하거나 기술 문서를 작성해본 경험
